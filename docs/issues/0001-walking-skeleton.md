# 0001 歩く骨格: プラグイン登録と空ボードページ

Type: AFK

## What to build

プラグインの統合基盤を一気通貫で立ち上げる薄い縦スライス。Redmine にプラグインとして認識され、プロジェクトで有効化でき、専用権限を持つロールだけがプロジェクトメニューの「カンバン」タブから空のボードページに到達できる、という end-to-end の経路を通す。

- `init.rb` でプラグインを登録（名前・著者・バージョン・対象 Redmine 6.0+ を明記）。
- project module `:ez_kanban` を登録（プロジェクト単位で有効/無効化可能）。
- プロジェクト権限 `:view_ez_kanban` を登録（モジュール配下、ロールに付与可能）。
- ルートとコントローラ `KanbanController#show`（プロジェクトスコープ）。
- `:project_menu` に「カンバン」タブを追加（モジュール有効かつ権限保有時のみ表示）。
- 空のボードページを ERB で描画（見出しのみ、カードはまだ無し）。
- 全 UI 文字列は最初から `l()` 経由とし `ja.yml`/`en.yml` に登録（i18n 規約の起点）。

## Acceptance criteria

- [ ] プラグインが Administration → Plugins に表示される
- [ ] プロジェクト設定でモジュール `ez_kanban` を有効化できる
- [ ] モジュール有効かつ `:view_ez_kanban` 保有時のみメニューに「カンバン」タブが出る
- [ ] 権限の無いユーザーがボード URL に直接アクセスすると 403
- [ ] ボードページが空の状態で表示される（500 にならない）
- [ ] ja/en で見出しが切り替わる
- [ ] コントローラのアクセス制御テスト（権限あり/なし、モジュール無効）が通る

## Blocked by

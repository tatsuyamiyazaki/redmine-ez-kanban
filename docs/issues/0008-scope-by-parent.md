# 0008 スコープ（親による絞り込み）

Type: AFK

## What to build

特定の親配下の作業だけにボードを絞る機能（R3）。選んだ親の全子孫リーフ（深さ不問）を、現在のフィルタと AND して表示する。

- スコープ選択は2系統: Redmine 標準のチケットオートコンプリートで親を選ぶ + カードのパンくず各階層クリックでその祖先にスコープ（0004 のパンくず利用）。
- 子孫リーフ集合は nested set（親の `lft..rgt` かつ同 `root_id`）で深さ不問に取得（R3-2）。
- スコープ条件は現在のフィルタ条件に重畳（AND）して適用（R3-3 / R7-4）。
- スコープ状態は URL パラメータ（例 `scope_issue_id`）に保持し、共有・リロードで復元。
- スコープ解除でフラット表示（全リーフ）に戻る（R3-4）。
- WIP 件数・上限・サブプロジェクト範囲ともスコープ適用後の集合に対して整合的に動く。

## Acceptance criteria

- [ ] 親を選ぶと、その配下の全子孫リーフ（深い階層含む）だけが出る
- [ ] パンくずの階層クリックで同じ親にスコープできる
- [ ] スコープがフィルタと AND される
- [ ] スコープ状態が URL に載り、共有・リロードで復元される
- [ ] スコープ解除で全リーフ表示に戻る
- [ ] WIP 件数がスコープ適用後の総数になる
- [ ] 子孫集合算出・AND 合成のテストが通る

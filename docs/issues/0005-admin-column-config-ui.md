# 0005 列設定の管理UI

Type: HITL

## What to build

管理者が列構成を編集できる設定画面を提供し、0003 で組込み既定に頼っていた列定義を実データで運用可能にする。設定エディタの UX 設計・レビューが必要なため HITL。

- Administration → Plugins → Configure（プラグイン設定 partial）で `Setting` のシリアライズ設定を編集。
- 編集対象: 列の追加/削除/リネーム、並び順、ステータス→列のマッピング、`is_done` 列の指定（1つ）、列ごとの WIP しきい値。
- 「未分類」列は暗黙・常時末尾で、管理者は編集不可（ADR-0004）。
- WIP しきい値による視覚強調を実装。既定オフ（R10-3）。しきい値超過は情報表示にとどめ、カードの配置や表示可否に影響しない（R10-4）。
- 設定は単一グローバル（全プロジェクト共通、R6-5）。不正・未マッピングは 0003 のフォールバックで安全に処理。

## Acceptance criteria

- [ ] 列の追加/削除/リネーム/並び替えができ、ボードに反映される
- [ ] ステータス→列マッピングを編集でき precedence 通り反映される
- [ ] `is_done` 列を1つ指定できる
- [ ] 列ごとに WIP しきい値を設定でき、超過列が強調される
- [ ] しきい値強調が既定オフで、超過してもカード配置/表示数は変わらない
- [ ] 「未分類」列は UI から編集できない
- [ ] 設定の保存/読込/バリデーションのテストが通る

## Blocked by

- 0003 列レイアウト・配置・WIP件数・列内ソート
